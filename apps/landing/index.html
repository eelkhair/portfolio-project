Clean Architecture + DDD

CQRS + Decorator Pipeline

Strangler-Fig Migration

AI Multi-Provider

OpenAI, Claude, and Gemini with scoped tool registries, MCP servers, and function calling.

Resume RAG Pipeline

Distributed Tracing

Transactional Outbox

Events written atomically with state changes. UPDLOCK/READPAST locking, retries, and dead-letter handling.
